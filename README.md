# Task Automation & Management System

## 🚀 Overview
This is a Python-based productivity system developed to manage tasks, track deadlines, and improve efficiency using automation.

## 🔧 Features
- User Login & Registration
- Task Management (Add, View, Complete, Delete)
- Priority & Deadline Tracking
- CSV Export
- Analytics using Matplotlib
- Flask Web Interface

## 🛠 Tech Stack
- Python
- Flask
- SQLite
- Pandas
- Matplotlib

## ▶ How to Run

```bash
pip install -r requirements.txt
python -m app.web_app
